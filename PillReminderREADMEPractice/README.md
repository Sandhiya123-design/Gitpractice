# Pill Reminder System

## Description

A healthcare application developed to help users remember and manage their medication schedules effectively.

## Features

- Add medicines
- Schedule reminders
- View medication list
- Manage patient records

## Technologies Used

- Python
- SQLite
- Tkinter

## Installation

1. Clone the repository
2. Open the project folder
3. Run the Python application

## Usage

Users can add medicines and receive reminders based on the scheduled timings.
